# Kütüphane Otomasyonu


# Installation

Öncelikle projeyi klonlayın.
```
git clone https://github.com/nursenyanar/kutuphane-otomasyonu
```
Eğer sinatra kurulu değilse sinatra kurulumu yapmak için:
```
gem install sinatra
```
Projeyi çalıştırmak için:
```
ruby mainpro.rb
```
